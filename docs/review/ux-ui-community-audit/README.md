# Rodada UX/UI + Pesquisa Comunidade

Pasta designada para relatorios independentes da rodada de avaliacao UX/UI do `lichess-tutor`.

Cada IA deve usar o prompt canonico:

- `prompts/ux-ui-community-audit.md`

E salvar seu relatorio neste padrao:

- `docs/review/ux-ui-community-audit/relatorio-NOME-DA-IA-ux-ui-comunidade-2026-06-07.md`

Objetivo da rodada: avaliar o funcionamento real da UX/UI atual e pesquisar componentes, bibliotecas,
testes e padroes prontos da comunidade que possam melhorar o app sem reinventar pecas dificeis.

As recomendacoes devem respeitar `AGENTS.md`: clean-room, sem tabuleiro proprio, sem ajuda em partida
ao vivo, sem scraping, sem escopos de jogo, e sem adiantar P4/P5.
