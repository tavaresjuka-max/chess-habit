## Notas Metodológicas

### Fontes que mais renderam

1. **Internet Archive** — melhor fonte para **livros clássicos** de xadrez. 303+ itens com licença confirmada. A busca por `subject:chess` é mais abrangente que `title:chess`. A maioria dos livros pré-1928 está em domínio público confirmado.

2. **OpenAlex** — melhor fonte para **papers acadêmicos** com PDF. 1762 papers com "chess" no título e acesso aberto confirmado. Necessita triagem (muitos usam "CHESS" como acrônimo não-relacionado).

3. **DOAJ** — boa fonte para **artigos de periódicos** open access. 1074+ artigos. A maior parte é de ciências (medicina, astronomia) usando CHESS como acrônimo, mas há artigos relevantes de educação.

4. **Gutendex (Project Gutenberg)** — API fora do ar durante toda a pesquisa (HTTP 503). Quando disponível, deve render 20-40 livros clássicos de xadrez em domínio público. Tentar novamente em outro momento.

5. **ERIC** — inacessível via webfetch desta rede. Pode ser consultado manualmente em https://eric.ed.gov/.

6. **OATD** — retornou HTTP 403. Pode ser consultado manualmente em https://oatd.org/.
