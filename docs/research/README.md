# Pesquisa

Use esta pasta como arquivo de evidencia e relatorios do projeto.

## Entrada Canonica

- `sources.md`: fontes oficiais, academicas e tecnicas ja consultadas.
- `method_synthesis.md`: sintese pedagogica limpa para orientar o metodo proprio.
- `curriculum_map.md`: mapa comparativo de metodos e blocos curriculares.
- `academic_evidence.md`: evidencia cognitiva e educacional.

## Relatorios De Apoio

Arquivos `analise-*`, `relatorio-*` e `plano-implementacao-*` sao insumos historicos de agentes.
Eles preservam rastreabilidade, mas nao substituem `PLANO.md`, `AGENTS.md`, `memory/` ou os documentos
canonicos em `docs/pedagogy/`.

## Acervo

- `chess-literature/`: manifestos e fichas de literatura.
- Downloads locais ficam fora do Git por regra de higiene e direitos autorais.
