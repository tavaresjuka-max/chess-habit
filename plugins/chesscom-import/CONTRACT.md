# Contrato: Chess.com Import

## Objetivo

Importar historico publico simples por username para estimar nivel inicial.

## Entrada

- Username Chess.com informado pelo aluno.

## Saida

- Sinais derivados de rating, ritmo, volume e atividade.

## Proibido

- Login Chess.com.
- Senha.
- Scraping.
- Equivalencia exata de rating.
- Importacao pesada sem necessidade.
