# Quality Gate

## Fase De Documentacao

Verificar:

- nao existe `package.json`;
- nao existe `src`;
- nao existe `node_modules`;
- documentos essenciais existem;
- aviso de app nao oficial esta no README e PLANO;
- contratos em `plugins/` sao documentais;
- nao ha conteudo do app pago anterior.

## Fase Futura De Codigo

Verificar:

- typecheck;
- lint;
- testes unitarios;
- testes de sync;
- teste PWA desktop/mobile;
- teste offline;
- revisao de privacidade.
