# Convencoes

## Linguagem

- Documentacao em PT-BR simples.
- Preferir frases curtas e decisoes explicitas.
- Evitar jargoes sem explicar impacto.

## Produto

- "Treino" significa atividade feita no Lichess.
- "Tutor" significa planejamento, acompanhamento, feedback e adaptacao.
- "Sync" significa progresso do tutor entre dispositivos.

## Dados

- Salvar o minimo necessario.
- Separar dado bruto de sinal derivado.
- Preferir sinal derivado a PGN completo.
- Exportacao e exclusao devem ser requisitos do MVP.

## Arquivos

- `memory/` guarda estado duravel.
- `docs/` guarda decisoes e pesquisa.
- `prompts/` guarda comandos copiaveis para revisao por IA.
- `skills/` guarda workflows para agentes.
- `plugins/` guarda contratos documentais de integracao futura.
